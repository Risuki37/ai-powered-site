# コンポーネントディレクトリ

## 構造

```
components/
├── ui/              # 基本UIコンポーネント（shadcn/ui等）
├── auth/            # 認証関連コンポーネント
├── blog/            # ブログ関連コンポーネント
└── todos/           # Todo関連コンポーネント
```

## 使い方

### UIコンポーネント

`components/ui/`には、shadcn/uiなどの再利用可能な基本UIコンポーネントを格納します。

### 機能別コンポーネント

各機能ごとにディレクトリを分けて、関連するコンポーネントを整理します。
